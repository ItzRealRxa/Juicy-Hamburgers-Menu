# Juicy Hamburgers Menu

A small HTML/CSS project showcasing a hamburger menu UI.

Live demo: https://itzrealrxa.github.io/Juicy-Hamburgers-Menu/

![Homepage screenshot](assets/screenshot.png)

## Features

- Responsive hamburger menu
- Pure HTML & CSS implementation

## How to open

- Open index.html in your browser, or visit the live demo above.
